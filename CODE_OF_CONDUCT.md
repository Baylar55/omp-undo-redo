# Code of Conduct

## Our standard

We want this project to be welcoming, respectful, and useful to everyone. Participants must communicate in good faith and focus on the work rather than people. Harassment, discrimination, personal attacks, sexualized conduct, doxxing, threats, and deliberately disruptive behavior are not acceptable.

## Expected behavior

- Be respectful and constructive.
- Give and receive feedback professionally.
- Respect different backgrounds, experiences, and viewpoints.
- Protect private information and project credentials.
- Prefer clear technical discussion and evidence over speculation.

## Enforcement

Report unacceptable behavior to the project maintainers through the repository's private security/reporting channel. Do not include sensitive information in a public issue. Maintainers may remove content, limit participation, or take other action appropriate to the circumstances.

This code of conduct is adapted from the Contributor Covenant. The project maintainers are responsible for interpreting and enforcing it.
